Setup:
Add the JDBC driver to your project.

    Download the PostgreSQL JDBC driver from https://jdbc.postgresql.org/download/
    Add the .jar file to your project as an external library.

Modidy these lines to match your postgresql server/database infromation:
    private final String url = "jdbc:postgresql://localhost:5432/3005a3q1";
    private final String user = "postgres";
    private final String password = "postgres";

to run:
make sure you have the vscode java extension pack installed
and then run java on the DatabaseOperations.java file
